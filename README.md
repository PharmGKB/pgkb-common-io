This package contains common IO code shared among all PharmGKB projects.

## Get It

### Maven

```xml
<dependencies>
  ...
  <dependency>
    <groupId>org.pharmgkb</groupId>
    <artifactId>pgkb-common-io</artifactId>
    <version>0.2.0</version>
  </dependency>
  ...
</dependencies>
```
